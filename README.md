# CodeAlpha_Hangman-Game
This is a simple text-based Hangman game built using Python as part of the CodeAlpha Python Programming Internship.

## Project Description
The Hangman game randomly selects a word from a predefined list.  
The player guesses the word one letter at a time. The player has a maximum of 6 incorrect guesses before the game ends.

## Features
- Random word selection
- Displays guessed and hidden letters
- Maximum of 6 incorrect attempts
- Simple console-based gameplay

## Technologies Used
- Python 3
- Random module
